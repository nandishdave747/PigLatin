# Pig Latin Translation Microservice
Convert paragraph to Pig Latin

Step - 1: Install Flask using "pip install flask"

Step - 2: Run piglatin.py file using python "piglatin.py"

Step - 3: Microservice Structure

##URL: 
```/piglatin```

##Method: 
```POST```

## Request Body Format

```
{
  "data": String
}
```
